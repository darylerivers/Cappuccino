## What to Avoid on eBay

### ‚ùå Never Buy These:

**1. Consumer Desktop Drives**
- WD Blue, Seagate Barracuda, Toshiba DT01
- Not built for 24/7 operation
- High failure rates
- Often SMR technology

**2. Surveillance Pulls**
- "Pulled from DVR/NVR system"
- Usually WD Purple, Seagate SkyHawk
- Very high write cycles
- Optimized for video, not backups

**3. Laptop/2.5" Drives**
- Too slow (5400 RPM)
- Lower capacity
- Not suitable for backups

**4. Drives Without SMART Data**
- Seller refuses to provide SMART report
- Red flag - hiding something
- Skip these listings

**5. Too-Good-To-Be-True Prices**
- 4TB for $20 = scam
- 8TB for $40 = scam
- If 50%+ below market, it's fake

**6. New Sellers or Low Ratings**
- < 98% positive rating
- < 100 transactions
- Account < 6 months old
- Higher scam risk

**7. "Untested" or "As-Is"**
- Seller didn't test = broken
- No returns = stuck with broken drive
- Not worth the risk

## Reading SMART Data

### Key SMART Attributes to Check

**Most Important:**
1. **Power_On_Hours (09):**
   - < 10,000 hours: Excellent
   - 10,000-20,000: Good
   - 20,000-40,000: Acceptable
   - > 40,000: Risky (but cheap)
   - > 60,000: Avoid

2. **Reallocated_Sector_Ct (05):**
   - 0: Perfect ‚úÖ
   - 1-10: Concerning ‚ö†Ô∏è
   - > 10: Avoid ‚ùå

3. **Current_Pending_Sector (197):**
   - 0: Good ‚úÖ
   - > 0: Drive is failing ‚ùå

4. **Offline_Uncorrectable (198):**
   - 0: Good ‚úÖ
   - > 0: Drive has errors ‚ùå

**Calculation:**
- Power-on hours √∑ 24 √∑ 365 = years of use
- 20,000 hours = 2.3 years
- 40,000 hours = 4.6 years

**Example good SMART:**
```
  5 Reallocated_Sector_Ct   0
  9 Power_On_Hours          18500  (2.1 years)
197 Current_Pending_Sector  0
198 Offline_Uncorrectable   0
```

**Example bad SMART:**
```
  5 Reallocated_Sector_Ct   45     ‚Üê BAD!
  9 Power_On_Hours          65000  ‚Üê OLD!
197 Current_Pending_Sector  12     ‚Üê FAILING!
198 Offline_Uncorrectable   8      ‚Üê ERRORS!
```

## Testing After Purchase

### Day 1: Initial Test
```bash
# Check SMART data
sudo smartctl -a /dev/sdb

# Look for:
# - Power-on hours matches listing
# - 0 reallocated sectors
# - 0 pending sectors
# - No errors
```

### Day 1-3: Short Test
```bash
# Run SMART short test (5 minutes)
sudo smartctl -t short /dev/sdb

# Wait 5 min, check results
sudo smartctl -a /dev/sdb
```

### Day 3-7: Full Surface Test
```bash
# Run extended test (8-12 hours)
sudo smartctl -t long /dev/sdb

# Or use badblocks (more thorough)
sudo badblocks -wsv /dev/sdb
# WARNING: This erases all data!
```

### If Any Issues:
- **Within 30 days:** Return via eBay
- **Contact seller first:** Give them chance to resolve
- **Open eBay case if needed:** eBay usually sides with buyers
- **PayPal protection:** 180 days
